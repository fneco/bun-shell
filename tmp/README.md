# tmp

To install dependencies:

```bash
bun install
```

To run:

```bash
# in a terminal
bun run launch.ts
# then, move to specific page.

# in another terminal
bun run html.ts
bun run index.ts
```

This project was created using `bun init` in bun v1.0.25. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
